# Fast React Pizza

## Overview

**Fast React Pizza** is a web application designed to streamline the process of ordering pizzas. Built with React and Vite, it offers a fast and responsive user experience, allowing customers to customize their pizza orders and track them in real-time.

## Features

- **Pizza Customization**: Choose from a variety of toppings, crust types, and sizes to create the perfect pizza.
- **Real-Time Order Tracking**: Monitor the status of your order from preparation to delivery.
- **User Authentication**: Secure sign-up and login functionality for a personalized experience.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Linting**: ESLint
- **Formatting**: Prettier

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/melvinprince/fast-react-pizza.git
   cd fast-react-pizza
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Start the Development Server**:

   ```bash
   npm run dev
   ```

   The application will run on `http://localhost:3000`.

## Usage

1. **Browse Menu**: Explore the variety of pizzas available.
2. **Customize Your Pizza**: Select your preferred toppings, crust, and size.
3. **Add to Cart**: Review your selections and add them to your cart.
4. **Checkout**: Complete the purchase by providing delivery details and payment information.
5. **Track Your Order**: Use the real-time tracking feature to monitor your pizza's journey.
